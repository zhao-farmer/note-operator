# 四、其他配置

1. 自定义Css

可以通过覆盖根级别的 CSS 变量来自定义默认主题的 CSS：

```js
// .vitepress/theme/index.js
import DefaultTheme from 'vitepress/theme'
import './custom.css'

export default DefaultTheme
```


```css
/* .vitepress/theme/custom.css */
:root {
  --vp-c-brand-1: #646cff;
  --vp-c-brand-2: #747bff;
}
```

2. markdown 目录配置

```js
export default{
  themeConfig: {
    ...
    // 右边目录是否关闭
    aside:true,
    // 右边目录配置
    outline:{
      level:[2,4],
      label:'目录'
    },
  }
}
```

3. 设置所有markdown文件所在目录

```js
export default{
	// markdown所在目录
	srcDir: './src',
}
```
