# 二、使用Cloudflare部署NUXT4

## 2.1 准备工作 

1. 概述

    [Cloudflare](https://dash.cloudflare.com/) 是一家全球化的云服务公司，提供网络安全、内容分发网络（CDN）、域名解析（DNS）、DDoS防护等基础设施服务。其核心目标是通过分布式网络优化网站性能并增强安全性。

2. 登录账户

    Cloudflare可以使用谷歌账号，Apple账号和github账号一键登录，任选一个点击进入授权登录即可。开发人员就直接用Github 账号授权登录即可。

    ![](/other/install/cloudflare/001.png)

3. nuxt.config.ts 配置

    ```ts
    export default defineNuxtConfig({
        compatibilityDate: '2024-11-01',
        nitro: {
            preset: 'cloudflare-pages'
        },
        // 如果使用 SSG
        // ssr: true  // 默认就是 true
        
        // 如果使用静态生成
        // ssr: false
    })
    ```
4. 本地构建测试

    ```
    # 安装依赖
    npm install

    # 构建
    npm run build

    # 预览构建的功能
    npm run preview
    ```
