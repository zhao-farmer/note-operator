# 十、快速答题


1. java 集合扩容 
    ArrayList底层是Object[]数组，扩容机制有三点关键：
    
    1. 默认初始容量是10
    2. 扩容因子是1.5倍（newCapacity = oldCapacity + (oldCapacity >> 1)）
    3. 扩容时通过Arrays.copyOf创建新数组，复制原数据
    时间复杂度O(n)，涉及数组复制"

2. 异常处理

    Java异常分为两大类、三小类：

    ```
    🔴 Error（系统级，不可恢复）
    ├─ OutOfMemoryError
    ├─ StackOverflowError
    └─ VirtualMachineError
    🟡 Exception（应用级）
    ├─ 受检异常（Checked，必须处理）
    │   ├─ IOException
    │   ├─ SQLException
    │   └─ ClassNotFoundException
    └─ 运行时异常（Unchecked，可处理）
        ├─ NullPointerException
        ├─ IllegalArgumentException
        ├─ IndexOutOfBoundsException
        └─ 业务自定义异常
    ```
