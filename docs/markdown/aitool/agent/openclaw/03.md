# 三、接入其他应用

## 3.1 接入飞书

### 3.1.1 飞书设置

1. 进入飞书开发者后台，创建应用

    网站：[https://open.feishu.cn/app](https://open.feishu.cn/app)

    ![](/aitool/agent/openclaw/024.png)

2. 选项设置

    ![](/aitool/agent/openclaw/025.png)

3. 权限设置

    1. 选择开通权限

        ![](/aitool/agent/openclaw/027.png)
    2. 选择应用权限

        1. `获取与上传图片或文件资源`
        2. `获取用户基本信息`
        3. `获取群组信息`
        4. `获取与发送单聊、群组消息`
        5. `接收群聊中@机器人消息事件`
        6. `获取群组中所有消息`(敏感权限)
        7. `读取用户发给机器人的单聊消息`
        8. `查看消息表情回复`
        9. `获取单聊、群组消息`
        10. `以应用的身份发消息`
        11. `更新消息`

    3. 确认11项，并开通权限

        ![](/aitool/agent/openclaw/028.png)

        ![](/aitool/agent/openclaw/029.png)

4. 复制凭证信息

    ![](/aitool/agent/openclaw/030.png)

    先保存到临时文件中，后续需要使用

5. 创建版本信息

    ![](/aitool/agent/openclaw/031.png)

    ![](/aitool/agent/openclaw/032.png)

    ![](/aitool/agent/openclaw/033.png)


### 3.1.2 openclaw 设置

1. 输入命令添加聊天工具

    ```sh
    openclaw channels add
    ```

2. 命令展示

    - 选择飞书

        ![](/aitool/agent/openclaw/034.png)
    - 选择安装插件

        ![](/aitool/agent/openclaw/035.png)
    - 输入飞书上的密钥

        ![](/aitool/agent/openclaw/036.png)
    - 飞书连接方式-websocket

        ![](/aitool/agent/openclaw/037.png)

    - 再次进入连套选择，直接选择完成

        ![](/aitool/agent/openclaw/038.png)

### 3.1.3 回调设置

1. 重启openclaw 网关

    ```sh
    openclaw gateway restart
    ```

    出现下面的信息，算作成功

    ![](/aitool/agent/openclaw/041.png)

2. 事件与回调（选择长连接）

    ![](/aitool/agent/openclaw/039.png)

3. 创建新版本

    ![](/aitool/agent/openclaw/040.png)
