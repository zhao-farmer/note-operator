# 一、cuda的安装与使用

## 1.1 安装CUDA（以主流方式为例）

CUDA的安装方式有多种，您可以根据自己的操作系统和使用习惯选择最合适的一种。

### 1.1.1 **Windows系统**

在Windows上，最直接的方式是使用NVIDIA提供的**本地或网络安装包**。

1.  **下载**：访问 [CUDA Toolkit下载页面](https://developer.nvidia.com/cuda-downloads)，选择您的Windows版本和安装包类型。本地安装包（Local）较大但可离线安装，网络安装包（Network）较小但需在安装时下载所需组件。

2.  **安装**：双击下载的安装包，按照屏幕提示操作即可。安装过程会包含CUDA驱动（如果未安装或版本较旧）、CUDA runtime库、开发工具（如NVCC编译器）等核心组件。

3.  **其他方式**：
    *   **Conda**：如果您是Python用户，可以使用Conda进行安装，它能更好地管理环境依赖：`conda install cuda -c nvidia`。
    *   **Pip Wheels**：NVIDIA也提供了Python wheels包，方便在Python环境中直接安装CUDA runtime：`py -m pip install nvidia-cuda-runtime-cu12`。

### 1.1.2 **Linux系统**

Linux下的安装方式更为灵活，主要有以下几种：

1.  **包管理器安装（推荐）**：
    *   这是最推荐的方式，因为它会自动处理依赖关系，便于后续的更新和卸载。
    *   对于**Ubuntu/Debian**系统，可以使用`apt`命令：
        ```bash
        sudo apt update
        sudo apt install nvidia-cuda-toolkit
        ```
        
    *   对于**Red Hat/CentOS**系统，可以使用`yum`或`dnf`。首先需要添加NVIDIA的官方仓库。
2.  **Runfile安装**：
    *   这种方式提供了一个独立的、与发行版无关的安装包，适合需要高度定制化安装的场景。
    *   首先从官网下载 `.run` 文件。然后，执行以下命令开始安装：
        ```bash
        chmod +x cuda_<version>_linux.run
        sudo ./cuda_<version>_linux.run
        ```
        需要注意的是，在安装过程中，如果您的系统**已经安装了NVIDIA显卡驱动，务必取消勾选`Driver`选项**，以避免驱动冲突。

## 1.2 验证与查看CUDA安装

安装完成后，需要通过几个简单的命令来确认CUDA是否安装成功以及版本信息。

1.  **查看CUDA编译器版本**：
    在命令行中输入 `nvcc --version` 或 `nvcc -V`。如果安装成功，您将看到CUDA编译器的版本号，这代表了您安装的CUDA Toolkit的版本。
    ```bash
    nvcc --version
    ```

2.  **查看NVIDIA驱动信息和运行时CUDA版本**：
    在命令行中输入 `nvidia-smi`。这个命令会列出您GPU的详细信息、NVIDIA驱动版本，以及**Driver API支持的CUDA版本**。如下图所示，右上角的"CUDA Version"表示您的驱动最高可以支持到这个版本的CUDA。

3.  **运行官方示例程序（可选，强烈推荐）**：
    这是验证CUDA功能完整性的最佳方法。您可以下载NVIDIA提供的CUDA示例代码，编译并运行`deviceQuery`和`bandwidthTest`这两个小程序。
    *   在**Windows**下，您需要找到CUDA Samples目录（如 `C:\ProgramData\NVIDIA Corporation\CUDA Samples`），打开其中的Visual Studio解决方案文件（如 `Samples.sln`），在Visual Studio中编译整个解决方案。然后，在命令行中进入到编译输出的目录（如 `\bin\win64\Release`），分别运行 `deviceQuery.exe` 和 `bandwidthTest.exe`。
    *   在**Linux**下，您可以通过以下命令操作：
        ```bash
        # 进入示例程序编译目录（请根据您的CUDA安装路径调整）
        cd /usr/local/cuda/extras/demo_suite/
        # 运行deviceQuery
        ./deviceQuery
        # 运行bandwidthTest
        ./bandwidthTest
        ```
    如果这两个程序的运行结果最后都显示 **`Result = PASS`**，那么恭喜您，CUDA已经成功安装并可正常使用。
