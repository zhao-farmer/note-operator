# 网络相关

## 2.1 添加域名解析

1. 购买域名

    以腾讯云服务为例： [https://cloud.tencent.com/product/domain](https://cloud.tencent.com/product/domain)

    ![](/operations/simulate/cloud/010.png)

    购买后大概需要等待1-3天才会成功

2. 云服务器添加DNS解析
    
    添加域名解析记录：‌ 首先登录腾讯云控制台，进入“云解析DNS”服务；如果域名未在腾讯云注册，需先单击【添加域名】将其加入管理列表。随后选择域名并点击【解析】，在记录管理页面单击【添加记录】，常见配置包括：
    
    - 记录类型‌：选择“A记录”指向IPv4地址，或“CNAME记录”指向另一个域名。
    - 主机记录‌：填写“@”代表域名根（如example.com），“www”代表子域名（如www.example.com）。
    - 记录值‌：输入腾讯云服务器的公网IP地址（可通过云服务器控制台查看）。
    - TTL‌：默认600秒，表示DNS缓存时间。

    ![](/operations/simulate/cloud/011.png)

    > 注意：不会立刻生效

    ![](/operations/simulate/cloud/012.png)
