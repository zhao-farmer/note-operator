# 一、云服务器与购买服务器

## 1.1 主要应用场景

1. 网站与应用托管
    云服务器是网站托管的理想选择，无论是个人博客、企业官网还是电商平台，都能提供可靠的支持。通过将网站部署在云服务器上，用户可以获得更好的性能和可扩展性，特别是在流量高峰期可以根据需求快速增加资源。

2. 电商与高并发业务
    
    电商平台、抢票系统等需要处理高并发访问的应用，可以利用云服务器的弹性伸缩功能。平时使用低配置实例节省成本，在活动前自动扩展为高配置实例，流量高峰过后再缩回，按秒计费不浪费资源。
3. 大数据与AI应用

    云服务器为大数据分析和人工智能应用提供了强大的计算能力和存储空间。企业可以利用云服务器进行大规模数据处理、深度学习模型训练等任务，按需使用GPU资源，大大降低了AI应用的门槛。
4. 开发与测试环境

    在软件开发过程中，云服务器提供了便利的开发和测试环境。开发人员可以快速创建和删除测试环境，而无需承担物理服务器的购置和维护成本，支持DevOps和持续集成流程。
5. 数据备份与灾备
    
    云服务器支持自动备份和灾难恢复功能，企业可以定期将关键数据备份到云端，防止因硬件故障或自然灾害导致的数据丢失。在发生系统故障时，可以通过云服务器快速恢复业务，确保业务连续性
## 1.2 主要厂商对比

1. 全球市场格局

    全球云服务器市场已形成"3+N"竞争格局，AWS、Azure、阿里云占据超60%市场份额，谷歌云、腾讯云、华为云等紧随其后。
2. 国内主要厂商
    - 阿里云：市场份额领先，在电商、金融级应用领域优势明显，提供金融级SLA稳定性
    - 腾讯云：在游戏、音视频、社交领域优势显著，具备高并发处理能力
    - 华为云：在政企、物联网领域影响力显著，通过等保认证，适合对安全要求高的企业
    - 百度云：AI能力强大，在图像识别、语音识别等AI应用场景丰富
    - 京东云：在电商领域优势明显，提供多元化的云计算服务

3. 选型建议
    - 追求全能型选手、高性价比：选择阿里云
    - 游戏社交、微信生态：选择腾讯云
    - 政企客户、物联网/制造业：选择华为云

## 1.4 测试nginx

1. 安装linux

    - 命令安装

        ```sh
        # 安装nginx
        yum install nginx
        # 启动nginx
        systemctl start nginx
        ```
    - 直接公网IP调用

        ![](/operations/cloud/002.png)

2. 修改nginx中数据

    - 查看nginx位置

        ```sh
        # 查看nginx位置
        whereis nginx
        ```
    - 修改html文件

        ```sh
        # 进入html页面
        cd /usr/share/nginx/html
        # 输出数据到页面
        echo "666" > index.html
        ```
    - 运行结果

        ![](/operations/cloud/003.png)

## 1.5 安装组设置

1. 端口无法访问

    - 设置ngnix端口号

        ```sh
        # 查看nginx位置
        whereis nginx
        # 进入安装位置
        cd /etc/nginx
        # 修改配置文件  80改为88
        vim nginx.conf
        # 重启服务
        systemctl restart nginx
        ```
