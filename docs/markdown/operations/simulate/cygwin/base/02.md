8. 选择安装包

    - 选项中base就是基础安装包

![](/operations/simulate/cygwin/base/014.png)

2. 选择需要的安装包

![](/operations/simulate/cygwin/base/015.png)

## 2.4 检测组件包

1. 查看所有已安装的Cygwin包

```sh
cygcheck -c
```

2. 搜索特定的包

```sh
cygcheck -c | grep <包名>
```
