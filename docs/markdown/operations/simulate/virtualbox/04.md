
## 四、常见问题与注意事项

1.  **虚拟机无法识别网卡？**  
    重启虚拟机即可（VirtualBox添加网卡后需重启才能识别）。
2.  **ping百度失败？**  
    检查NAT网卡是否“启用网络连接”，或`ONBOOT=yes`是否配置正确。
3.  **主机ping不通虚拟机？**  
    确认Host-Only网卡的IP在`192.168.56.1-199`范围内，且虚拟机防火墙未拦截（新手可临时关闭防火墙：`systemctl stop firewalld`）。
4.  **vi编辑文件后无法保存？**  
    需用root用户操作（确保登录的是root，而非普通用户vboxuser）。
