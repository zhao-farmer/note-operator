# 卸载与删除软件

## 1. 卸载leagsoft安全卫士

之前公司装了一个安全卫士，是联软（leagsoft）公司的UniAccess Agent。极度流氓。

网上找了好多方法都不好使。下面直接说好使的方法

1.还在公司内部，那就正常卸载，联系管理员，给卸载密码就完事了

2.用winpe卸载。

1）用老毛桃，或者其他什么U盘启动，打开winpe

2）在winpe中找到C:\Windows\LVUAAgentInstBaseRoot 这个文件夹，彻底删除

3）然后就完事了~~~

## 2. 解决无法删除node_modules的问题

```sh
# 使用 rimraf（Node.js 工具）
npx rimraf node_modules
# 或先安装
npm install -g rimraf
rimraf node_modules

# 使用 nuke（更强大）
npx npkill
# 然后选择要删除的 node_modules
```
